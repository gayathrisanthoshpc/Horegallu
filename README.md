# Horegallu

A wellness app for anyone feeling overwhelmed. No login, no tracking, no data stored.

**Live:** https://horegallu.vercel.app

---

## What it has

- **Talk** — type whatever is on your mind, conversation disappears when you leave
- **Night mode** — a quieter space designed for late-night thoughts (10pm–5am)
- **Burnout check** — a 16-item honest check-in, no judgment
- **Unsent letter** — write something you never plan to send
- **Smallest step** — figure out the one small thing you can do right now
- **Multilingual** — available in English, Malayalam, and Hindi

## Built with

- Vanilla HTML, CSS, JavaScript
- No frameworks, no backend, no database
- Deployed on Vercel

## Run locally

Just open `index.html` in a browser. No setup needed.

```
git clone https://github.com/gayathrisanthoshpc/Horegallu
cd Horegallu
open index.html
```
